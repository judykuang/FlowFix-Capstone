# FlowFix

## Product Vision
Whether you're a plumber who is tired of larger corporations taking a big chunk of your pay or someone in need of a proper plumbing emergency fix, Flowfix is for you! Our AI-powered tool is designed to connect those on either end of the spectrum to their best match. It's best suited for homeowners, renters, landlords, realtors, and property managers.


### System Architecture Layers
1. **Presentation Layer**: HTML/CSS/JS frontend 
2. **Backend API Layer**: Node.js
3. **External AI API Layer**: Gemini API
4. **Data Access Layer**: via Node.js using an ORM/ODM 
5. **Database Layer**: NoSQL
